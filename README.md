# 게시판 구현 프로젝트
### 사용기술: java, springboot, Mybatis SQL Mapper / RDB: mariadb / template engine: Thymeleaf
#### 프로젝트 설명:   
CRUD를 기반으로 게시글 리스트,등록,수정,삭제를 할 수 있는 게시판 기능을 구현하였습니다.   
게시글 등록 시 파일을 업로드 할 수 있으며 재귀 함수를 사용하여 답글,댓글을 구현하였습니다.   
#### 게시판 기능
1. 게시글 등록
2. 게시글 수정
3. 게시글 삭제
4. 게시글 답글
5. 파일 업로드
6. 파일 삭제
7. 게시글 내 댓글 작성
8. 댓글 수정
9. 댓글 삭제
10. 대댓글 기능
